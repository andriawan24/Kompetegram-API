# Kompetegram-API
Dibuat untuk memenuhi sebagai anggota tim research di Kompetegram UPI Cibiru

### Project Setup
```
npm install
```

### Menjalankan project
```
node index.js
```

### Dokumentasi
Silahkan buka ```http://localhost:3000/api-docs``` untuk melihat dokumentasi

### Tech Stack
* MongoDB
* NodeJS
* ExpressJS
* Swagger
